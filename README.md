## Cubic Experiment

**Cubic Experiment** is a 3D puzzle game for web. Play on itch.io: https://jani-nykanen.itch.io/cubic-experiment

----

### Building

Run
```
make js
```
to build everything, and
```
make dist
```
to make a distributable zip file that contains everything.
